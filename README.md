# HR Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes employee attrition and workforce insights.

## KPIs
- Overall Employees
- Attrition Count
- Attrition Rate
- Active Employees
- Average Age

## Visualizations
- Department-wise Attrition
- Age Group Analysis
- Job Satisfaction Ratings
- Education Field-wise Attrition
- Gender-wise Attrition Analysis

## Tools Used
- Power BI
- DAX
- Excel
